CompositeStreamController

object CompositeStreamController extends Controller

An example standalone module that represents the "Who's Viewed Your Profile" module from the LinkedIn homepage. It's identical to the ProfileViewsController controller, except that the data is streamed to the browser as soon as it's available in small "pagelets", similar to Facebook's BigPipe.
